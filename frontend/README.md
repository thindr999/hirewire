# HireWire Frontend

This is the frontend application for HireWire.

## Features
- Business can book staff (Uber-style)
- Staff can register (primary or secondary employment)
- Availability calendar
- Ratings & reviews
- In-app payments
- Multi-language support

## Tech Stack
- React (Next.js)
- Tailwind CSS
- Stripe (payments)
- Firebase/Auth0 (authentication)
