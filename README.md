# HireWire

HireWire is an on-demand hospitality staffing platform (Uber-style) where:

- Businesses can book verified hospitality staff
- Staff can set availability and accept or decline shifts
- All certificates, experience, and ratings are visible
- Secure in-app payments
- Multi-language support
- Admin verifies staff before activation

## Roles
- Client (Business)
- Staff (Primary or Secondary Employment)
- Admin (Verification & Compliance)
